### üì¶ Getting Started

For start to develop with this project follow the steps below:

1. Install the dependencies (Use **npm** preferably):

    ```bash
    npm install
    ```

    Or if you prefer to use **yarn**:

    ```bash
    yarn install
    ```

2. Run the application in mode `design`:

    ```bash
    npm run storybook
    ```

    Or if you prefer to use **yarn**:

    ```bash
    yarn run storybook
    ```

> Note. This project has Storybook to help you to develop and test your components. You can join to first builds in Chromatic.
